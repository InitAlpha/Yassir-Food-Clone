# Getting started

1. [Setup Flutter](https://flutter.dev/docs/get-started/install)

2. Clone the repo

    ```
    $ git clone https://github.com/lagripe/Yassir-Food-Clone
    $ cd Yassir-Food-Clone/
    ```
3. Run
        `$ flutter run`
